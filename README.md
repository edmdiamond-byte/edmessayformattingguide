EDM Essay Formatting Lab
This academic writing reference tool assists in properly formatting academic papers.
It offers easy access to formatting guidelines so you can focus on research and idea development. 
Features:
•	Quick-reference for the APA 7th edition and MLA 9th edition academic writing rules
•	Can be installed on mobile and desktop devices
•	Once installed on your device, it can be used offline as well
•	Perfect as a final reference before submitting your draft for grading
•	Benefits anyone interested in writing scholarly papers
